## How to Use

1. **Install [Obsidian](https://obsidian.md/)** (free)
2. Clone this repo and open the folder as an Obsidian vault
3. Start at `00 - Home.md` — the master index linking to all sections
4. Each folder contains a `_MOC` (Map of Content) file as the entry point for that topic
