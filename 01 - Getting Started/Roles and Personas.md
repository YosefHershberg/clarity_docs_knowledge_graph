# Roles and Personas

Clarity is designed for various roles across the organization. Each persona interacts with different parts of the system.

## Key Personas

### Executive / Business Leader
- **Focus**: Strategic alignment, portfolio visibility, ROI
- **Primary workspaces**: [[_MOC Roadmaps|Roadmaps]], [[_MOC Hierarchies|Hierarchies]], [[Working with Objectives|Objectives]]
- **Key activities**: Review roadmaps, approve investments, monitor portfolio health

### Portfolio Manager
- **Focus**: Investment prioritization, resource optimization, financial oversight
- **Primary workspaces**: [[_MOC Investments|Investments]], [[_MOC Hierarchies|Hierarchies]], [[_MOC Reporting|Reports]]

### Project Manager
- **Focus**: Project delivery, task management, team coordination
- **Primary workspaces**: [[_MOC Projects|Projects]], [[Tasks Phases Milestones|Tasks]], [[Status Reports|Status]]
- **Key activities**: Create/manage projects, assign tasks, publish status reports

### Resource Manager
- **Focus**: Staffing, capacity planning, utilization
- **Primary workspaces**: [[_MOC Resources|Staffing]], [[Resource Directory]]

### Team Member
- **Focus**: Task completion, time tracking, collaboration
- **Primary workspaces**: [[_MOC Timesheets|Timesheets]], [[Action Items]], [[To Dos]]

### Financial Analyst
- **Focus**: Cost tracking, budgeting, forecasting
- **Primary workspaces**: [[_MOC Financials|Financials]], [[_MOC Reporting|Reports]]

### System Administrator
- **Focus**: Configuration, security, integrations
- **Primary workspaces**: [[_MOC Administration|Administration]], [[_MOC Install Upgrade|Installation]]

## Key Access Rights

| Right | Grants Access To |
|-------|-----------------|
| Project Management - Navigate | Projects grid view |
| Project Management - Tiles Navigate | Project Tiles view |
| Roadmap - Navigate | Roadmaps workspace |
| Idea - Navigate | Ideas workspace |

| Administration - Application | Administration pages |
