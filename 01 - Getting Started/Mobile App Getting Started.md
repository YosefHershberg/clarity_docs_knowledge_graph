# Getting Started with Clarity Mobile App

## Overview

The Clarity Mobile App provides on-the-go access to key features including timesheets, approvals, conversations, action items, and to-dos.

## Key Features

- **[[Timesheets Mobile|Timesheets]]** — Enter hours and submit timesheets from your phone
- **[[Review and Approve|Review & Approve]]** — Approve timesheets and other requests
- **[[Conversations Mobile|Conversations]]** — Participate in investment conversations
- **[[Action Items Mobile|Action Items]]** — View and manage action items
- **[[To Dos Mobile|To Dos]]** — Track and complete to-do items

## Getting Started

1. Download the Clarity app from the App Store (iOS) or Google Play (Android)
2. Enter your Clarity server URL
3. Sign in with your Clarity credentials
4. Navigate using the bottom tab bar

> [!tip]
> Contact your administrator for the correct server URL and any mobile-specific configuration requirements.
