# First Time Login Experience

When you log in to Clarity for the first time, a guided experience helps you get familiar with the interface.

## Accessing Clarity

1. Open your browser and navigate to the Clarity URL provided by your administrator
2. Enter your credentials (username and password)
3. Click **Sign In**

> [!note] Authentication Methods
> Clarity supports multiple authentication methods including LDAP, SAML SSO, and native authentication. See [[Authentication Methods]] for details.

## Main Navigation

The Clarity interface provides access to key workspaces through the main menu:

| Menu Item | Description |
|-----------|-------------|
| **Project Tiles** | Visual overview of projects as tiles |
| **Projects** | Grid view of all projects |
| **Ideas** | Idea management workspace |
| **Roadmaps** | Strategic planning workspace |
| **Timesheets** | Time entry and submission |
| **Staffing** | Resource allocation workspace |
| **Administration** | System configuration (admin only) |

## Key Interface Elements

- **[[Grids]]** — Spreadsheet-like data views used throughout Clarity
- **[[Board Layout]]** — Kanban-style drag-and-drop views
- **[[Widgets]]** — Visual analytics components on dashboards
- **[[Filters]]** — Filter data across all workspaces
- **[[Details Panel]]** — Quick-edit panel for record details
- **[[Saved Views]]** — Save and reuse custom view configurations

## Working with the AI Chat Panel

The AI Chat Panel allows you to interact with the AI Agent (Vaia) directly within Clarity:

1. Navigate to a workspace that supports the AI Agent
2. Click the **AI Agent logo** in the page header
3. Select an AI Chat Panel from the dropdown list

### Chat Panel Features
- **Follow-Up Prompts** — Predefined follow-up questions
- **User Input Text Box** — Type custom queries (if enabled)
- **Maximize/Dock/Minimize** — Adjust panel layout

> [!warning]
> Closing the chat panel ends the current interaction and clears conversation history. Each new AI Agent starts a fresh session.

## Register Your Instance

Navigate to **Administration > Register** to register your Clarity instance for support and updates.
