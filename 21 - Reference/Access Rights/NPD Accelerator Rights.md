# New Product Development (NPD) Accelerator Access Rights

The NPD Accelerator uses **access groups** rather than individual rights. Each group defines instance and global access rights for a specific user role in the NPD process.

## Access Groups

| Group | Purpose |
|---|---|
| Executive User Group | Executive/director level users: portfolio insight and gate scoring participation |
| Marketing Manager User Group | Manage ideas and facilitate movement through the scoring process |
| Non-Clarity User Group | Infrequent users who submit product ideas |
| Product Portfolio Manager User Group | Manage products and portfolios; participate in gate scoring |
| Project Portfolio Manager User Group | Manage projects and portfolios; participate in gate scoring |
| Team Member User Group | Resources assigned to projects/products; enter timesheets and upload documents |

## Executive User Group

**Instance Rights:**
- Page - View
- Process - Start (on each NPD Accelerator process)
- Project - Risk, Issue, Change Request - View (on the NPD Accelerator template)
- Project - View (on the NPD Accelerator template)
- Project - View Management (on the NPD Accelerator template)

- Gate Scoring - View All
- Idea - Hierarchy - Parents - View - All
- Idea - View All

- Portfolio - View All
