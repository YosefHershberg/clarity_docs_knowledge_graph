# PMBOK Guide Accelerator Access Rights

The PMBOK Guide Accelerator provides four access groups. Each group includes access rights for secure Classic PPM pages, dashboards, portlets, and documents.

## Access Groups Overview

| Group | Role |
|---|---|
| PBK Administrator Group | Administrators who set up Classic PPM and the PMBOK Guide Accelerator |
| PBK Executive Group | Executives who monitor progress, manage change, and approve documents |
| PBK Project Manager Group | Project managers who create, monitor, and manage projects |
| PBK Team Member Group | Team members who complete tasks and enter timesheets |

### Classic PPM Rights for Administrator Group

Includes Administration - Access, Audit Trail - View - All, Custom Objects Editor, Jobs - Access, Jobs - Run - All, Jobs - View Output - All, Knowledge Store - View All, Menu Definition Editor - All, Object - Create Definition, Object Administration, Object Mapping - XOG Access, Page Definition Editor - All, Page Viewer - All, Portlet Definition Editor - All, Portlet Viewer - All, Process rights (Cancel, Create Definition, Delete, Edit Definition, Manage, Start, View, All variants), Project rights (Approve - All, Create, Edit, Edit Management, Financial, View, View Management, etc.), Resource rights, and Timesheets rights.

## PBK Executive Group

### PMBOK Guide Accelerator-Specific Rights (all Global)

Includes Edit All and View All rights for: Lessons Learned, Product Description, Project Charter, Project Closure, Project Report, Project Scope Statement, Quality Management Plan, Risk Identification - SWOT Analysis, Scope Management Plan, and all corresponding secured subpage rights.

### Classic PPM Rights for Executive Group

| Right Name | Type | Description |
|---|---|---|
| Knowledge Store - View All | Global | Allows viewing all documents in the knowledge store. |
| Page - View | Instance | Allows viewing specific pages (Executive Dashboard, Lessons Learned, Project Charters, etc.). |
| Page Viewer - All | Global | Lets the user view any configured portlet page. |
| Portlet - View | Instance | Allows viewing specific portlets. |
| Portlet Viewer - All | Global | Lets you view and add stock portlets to portlet pages. |
| Project - Approve - All | Global | Allows approving all projects. |
| Project - Budget Plan - View All | Global | Allows viewing budget plans for all projects. |
| Project - Cost Plan - View All | Global | Allows viewing cost plans for all projects. |
| Project - Hierarchy - Financial Rollup - View - All | Global | Allows viewing financial rollup for all projects. |
| Project - Risk, Issue, Change Request - View - All | Global | Allows viewing all risks, issues, and change requests. |
| Project - View Financial - All | Global | Allows viewing general and financial properties for all projects. |
| Project - View Tasks - All | Global | Allows viewing tasks for any accessible project. |
| Project - View Management - All | Global | Allows viewing management properties for any project enabled for management. |
| Resource - Navigate | Global | Allows access to resource management pages. |
| Resource - View - All | Global | Allows viewing information for all resources (except financial). |
| Resource - View Book - All | Global | Allows viewing bookings for all resources. |
| Resource - View Financial - All | Global | Allows viewing financial properties for all resources. |

## PBK Project Manager Group

### PMBOK Guide Accelerator-Specific Rights (all Global)

Includes Create, Edit All, View All, and XOG Access for: Lessons Learned, Product Description, Project Charter, Project Closure, Project Report, Project Scope Statement, Quality Management Plan, Risk Identification - SWOT Analysis, Scope Management Plan.
Also includes all corresponding secured subpage rights (Edit All, View All, and some instance View rights on specific project templates).

### Classic PPM Rights for Project Manager Group

Includes all rights from the Executive Group plus: Project - Create, Project - Create from Template, Project - Edit Financial - All, Project - Edit Chargebacks Information All, Project - Enable Financial, Project - Reverse Charges - All, Project - Risk, Issue, Change Request - Edit - All, Resource - Hard Book - All, Resource - Edit - All, Resource - External Access, Resource - Soft Book - All, Timesheets - Approve All, Timesheets - Edit All.

## PBK Team Member Group

### PMBOK Guide Accelerator-Specific Rights (all Global - View only)

Includes View All rights for: Lessons Learned, Product Description, Project Charter, Project Closure, Project Report, Project Scope Statement, Quality Management Plan, Risk Identification - SWOT Analysis, Scope Management Plan, and all corresponding secured subpage View All rights.

### Classic PPM Rights for Team Member Group

| Right Name | Type |
|---|---|
| Page - View | Instance (Dashboard page) |
| Portlet - View | Instance (Project Checklist, Project Status, PMO Dashboard, Project Document Review Status) |
| Project - View Tasks - All | Global |
| Timesheets - Navigate | Global |
