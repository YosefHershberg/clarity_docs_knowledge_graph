# PMO Accelerator Access Groups

The PMO Accelerator includes access groups with global and instance-level access rights. Resources may be associated with more than one group (matrix assignment) to obtain all required rights.

> **Note:** Access rights for the Lookup Mapping object are only assigned to the PMO System Administrator group.

## Access Groups Overview

| Group | Purpose |
|---|---|
| PMO Executive Viewer | View all roadmaps, project properties, status reports, schedules, and resource allocations; run project and resource capacity reports |
| PMO Executive Viewer Extended | Same as Executive Viewer plus baselines and financials |
| PMO Financial Administrator | Manage project financial plans, create/edit financial transactions |
| PMO Idea Creator | Create and view new ideas |
| PMO Idea Reviewer | Review and approve ideas, including ability to edit all ideas |
| PMO Investment Creator | Create and view all investment types |
| PMO Investment Viewer Extended | View properties, resource allocations, and financial plans of all investment types |
| PMO Portfolio Manager | Manage portfolios across investments |
| PMO Program Manager | Manage programs containing several projects |
| PMO Project Creator | Create and view new projects |
| PMO Project Manager | Manage project properties, status reports, schedules, and resource allocations |
| PMO Project Manager Extended | Same as Project Manager plus tasks, baselines, and financial plans |
| PMO Project Viewer | View all projects and project-related portlets and reports |
| PMO Resource Administrator | Create resources/roles, edit properties, view all projects, adjust allocations |
| PMO Resource Manager | Manage schedules of direct reports, adjust allocations on projects, view all projects |
| PMO Resource Manager Extended | Same as Resource Manager plus skills, requisitions access |
| PMO System Administrator | Full global rights to all features and records |
| PMO Team Member | Create and submit timesheets |
| PMO Timesheet Administrator | Manage timesheet approval; submit and approve all timesheets |
| PMO Vendor Management | Access vendor-related reports and views |
| PMO Roadmap Creator | Create, view, and edit new roadmaps |
| PMO Roadmap Viewer | View all roadmaps |
| PMO Roadmap Editor | Edit all roadmaps and manage roadmap-specific picklists |

## PMO Executive Viewer

Key global rights: Projects - Navigate, Project Management - Navigate, Project Management - Tiles Navigate, Project - View Management - All, Project - Risk, Issue, Change Request - View - All, Status Report - View All, Resource - View - All, Resource - View Book - All, Project - View Requisitions - All, Roadmap - Navigate, Roadmap - View - All, Tasks - Navigate.

Also includes instance rights (Page - View and Portlet - View) on various resource planning, status report, timesheet, and project storyboard pages and portlets.

## PMO Executive Viewer Extended

Same as Executive Viewer plus: Project - View Financial - All, Project - Cost Plan - View All, Project - Budget Plan - View All, Project - Benefit Plan - View All, and access to Capacity Planning Overview and additional financial portlets.

## PMO Financial Administrator

Key rights: Project financial plan management, ability to create/edit any financial plan, approve budgets, create/edit financial transactions against any project.

Includes instance rights on Posted Transaction Review Page and related cost plan portlets.

Global rights include: Tasks - Navigate.

## PMO Idea Creator

Key global rights: Project Management - Navigate, Project Management - Tiles Navigate, Ideas - Create, Ideas - Navigate, Idea - View All, Resource - View - All, Tasks - Navigate.

## PMO Idea Reviewer

Key global rights: Same as Idea Creator plus Idea - Edit All, Idea - Approve All.

Includes instance rights on Sponsor Tab portlets.

## PMO Investment Creator

Key global rights: Creates rights for all investment types — Applications, Assets, Ideas, Other Work, Products, Services. Includes Navigate and Create rights for each.

## PMO Investment Viewer Extended

Provides view rights across all investment types:

- **Project Management:** Project - View Management - All, Project - View Financial - All, Project - Cost/Budget/Benefit Plan - View All, Project - Risk, Issue, Change Request - View - All, Status Report - View All
- **Application Management:** Application - View - All, Application - View Financial Information All, Application - Cost/Budget/Benefit Plan - View All
- **Asset Management:** Asset - View - All and financial view rights
- **Idea Management:** Idea - View - All and financial view rights
- **Other Work Management:** Other Work - View - All and financial view rights
- **Product Management:** Product - View - All and financial view rights
- **Service Management:** Service - View - All and financial view rights
- **Resource Management:** Resource - View - All, Resource - View Book - All, Project - View Requisitions - All

Also includes instance rights on resource planning, capacity planning, status report, timesheet, program status, and project portlet pages.

## PMO Portfolio Manager

Key global rights: Portfolio - Navigate, Portfolio - Create, Resource - View - All, Projects - Navigate, Project Management - Navigate.

Includes instance rights on Portfolio Dashboard pages with all portfolio portlets (Portfolio Dashboard, Filter, Costs, Role Demand, Investment Analysis, Capital and Operating, Portfolio Investment Dashboard).

## PMO Program Manager

Key global rights: Project - Enable Financial, Project - Cost/Budget/Benefit Plan - Edit All, Project - Budget Plan - Approval All, Project - Risk, Issue, Change Request - Edit/Delete - All, Status Report - Create, Status Report - Edit All, Management - Programs, Resource - View - All, Tasks - Navigate.

Includes instance rights on Program Dashboard, Issues and Risks, Status Report Review, and PMO Program/Project Status portlets.

## PMO Project Creator

Key global rights: Project - Create, Project - Create from Template, Project - Enable Financial, Resource - View - All, Tasks - Navigate.

## PMO Project Manager

Key global rights: Project - Enable Financial, Project - Modify Baseline All, Project - Risk, Issue, Change Request - Edit/Delete - All, Status Report - Create, Status Report - Edit All, Resource - View - All, Resource - Hard Book All, Tasks - Navigate.

Includes instance rights on Resource Planning, Status Report Review, Timesheet Overview, Timesheet Review, PMO-Project Storyboard pages, and associated portlets.

## PMO Project Manager Extended

All Project Manager rights plus: Project - Schedule In Browser, Project - Cost/Budget/Benefit Plan - Edit/Approve All, Resource - Soft Book All, Project - Create/Edit Requisitions - All, Project - Attach Requisitions Entry Resources - All.

Also includes access to Capacity Planning Overview, Sponsor Tab portlets, PM Alerts Tab, Project Dashboard Tab, and Issues and Risks Tab portlets.

## PMO Project Viewer

Key global rights: Project - View Management - All, Project - View Financial - All, Project - Risk, Issue, Change Request - View - All, Status Report - View All, Resource - View - All, Tasks - Navigate.

Includes instance rights on Project Dashboard, Issues and Risks, Status Report Review pages and associated portlets.

## PMO Resource Administrator

Key global rights: Resource - Navigate, Resource - Create, Resource - View - All, Resource - Edit - All, Resource - Edit Financial All, Resource - Update Skills - All, Team - Create, Team - Edit - All, Team - View - All, Team - Delete - All.

Includes instance rights on Resource Planning and Capacity Planning pages.

## PMO Resource Manager

Key global rights: Project - View Management - All, Project - View Financial - All, Resource - View - All, Resource - Hard Book All, Resource - Soft Book All, Tasks - Navigate.

Includes instance rights on Resource Planning and Capacity Planning pages and associated portlets.

## PMO Resource Manager Extended

All Resource Manager rights plus: Resource - Update Skills - All, Project - Create/Edit Requisitions - All, Project - Attach Requisitions Entry Resources - All, Project - View Requisitions - All.

Also includes instance rights on Timesheet Overview and Timesheet Review pages.

## PMO System Administrator

Members have all global rights to access all features and edit all records in the product.

## PMO Team Member

Key global rights: Projects - Navigate, Project Management - Navigate, Project Management - Tiles Navigate, Resource - View - All, Timesheets - Navigate, Tasks - Navigate.

Instance rights: Action Items, My Projects (portlets), My Timesheets.

## PMO Timesheet Administrator

Key global rights: Resource - View - All, Timesheets - Navigate, Timesheets - Edit All, Timesheets - Approval All.

Instance rights: Action Items, My Projects portlets.

## PMO Vendor Management

Key global rights: Projects - Navigate, Project Management - Navigate, Project Management - Tiles Navigate, Tasks - Navigate.

Instance rights: Action Items, My Projects portlets.
