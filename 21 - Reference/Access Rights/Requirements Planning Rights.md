# Requirements Planning Access Rights

Requirements Planning access rights cover Releases, Release Plans, and Requirements.

## Release Access Rights

| Right Name | Type | Description |
|---|---|---|
| Release - Approve | Instance | Allows the user to approve a specific release instance. |
| Release - Approve All | Global | Allows the user to approve all releases. |
| Release - Create | Global | Allows creating a release. Includes Releases - Navigate. Requires investment view rights for at least one investment. |
| Release - Edit | Instance | Allows editing a specific release instance. |
| Release - Edit All | Global | Allows editing all releases. |
| Release - Edit Access Rights | Instance | Allows editing access rights for a specific release. Does not include Releases - Navigate or Release - View. |
| Release - Edit Access Rights - All | Global | Allows editing access rights for all releases. Does not include Releases - Navigate or Release - View. |
| Release - View | Instance | Allows viewing a specific release instance. Does not include Releases - Navigate. |
| Release - View All | Global | Allows viewing all releases. Does not include Releases - Navigate. |
| Release - View Access Rights | Instance | Allows viewing access rights for a specific release. Does not include Releases - Navigate or Release - View. |
| Release - View Access Rights - All | Global | Allows viewing access rights for all releases. Does not include Releases - Navigate or Release - View. |
| Release - Manager (Auto) | Instance | Automatic right granted to the manager of a release. |
| Releases - Navigate | Global | Allows the user to access the release planning pages. |
| Release - XOG Access | Global | Allows importing and exporting release instances using XOG. |

## Release Plan Access Rights

| Right Name | Type | Description |
|---|---|---|
| Release Plan - Create | Global | Allows creating a release plan. Includes right to navigate to release plan pages. |
| Release Plan - View | Instance | Allows viewing a specific release plan instance. |
| Release Plan - View Access Rights | Instance | Allows viewing access rights for a specific release plan. Does not include Release Plan - Navigate or Release Plan - View. |
| Release Plan - Edit Access Rights | Instance | Allows editing access rights for a specific release plan. Does not include Release Plan - Navigate or Release Plan - View. |
| Release Plan - Edit Access Rights - All | Global | Allows editing access rights for all release plans. Does not include Release Plans - Navigate or Release Plan - View. |
| Release Plan - Edit | Instance | Allows editing a specific release plan instance. |
| Release Plan - Edit All | Global | Allows editing all release plans. |
| Release Plan - Navigate | Global | Allows the user to access the release planning pages. |
| Release Plan - View Access Rights - All | Global | Allows viewing access rights for all release plans. Does not include Release Plans - Navigate or Release Plan - View. |
| Release Plan - View All | Global | Allows viewing all release plans. |
| Release Plan - XOG Access | Global | Allows importing and exporting Release Plan instances using XOG. |

## Requirement Access Rights

| Right Name | Type | Description |
|---|---|---|
| Requirement - Approve | Instance | Allows the user to approve a specific requirement instance. |
| Requirement - Approve All | Global | Allows the user to approve all requirements. |
| Requirement - Create | Global | Allows creating a requirement. Includes right to navigate to requirement pages. |
| Requirement - Edit | Instance | Allows editing a specific requirement instance. Does not include Requirements - Navigate. |
| Requirement - Edit All | Global | Allows editing all requirements. |
| Requirement - Edit Access Rights | Instance | Allows editing access rights for a specific requirement. Does not include Requirements - Navigate or Requirement - View. |
| Requirement - Edit Access Rights - All | Global | Allows editing access rights for all requirements. Does not include Requirements - Navigate or Requirement - View. |
| Requirement - Manager (Auto) | Instance | Automatic right granted to the manager of a requirement. |
| Requirement - Prioritize | Instance | Allows editing the priorities subpage for a specific requirement. Does not include Requirements - Navigate. |
| Requirement - Prioritize All | Global | Allows prioritizing all requirements. Includes Requirement - Edit All. |
| Requirement - View | Instance | Allows viewing a specific requirement instance. Does not include Requirements - Navigate. |
| Requirement - View All | Global | Allows viewing all requirements. |
| Requirement - View Access Rights | Instance | Allows viewing access rights for a specific requirement. Does not include Requirements - Navigate or Requirement - View. |
| Requirement - View Access Rights - All | Global | Allows viewing access rights for all requirements. Does not include Requirements - Navigate or Requirement - View. |
| Requirements - Navigate | Global | Allows the user to access the requirements list page. |
