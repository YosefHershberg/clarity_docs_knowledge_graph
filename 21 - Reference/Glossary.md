# Clarity Glossary

Key terms and definitions used throughout Clarity PPM.

## Common Terms

| Term | Definition |
|------|-----------|
| **Investment** | Any project, idea, custom investment, or team in Clarity |
| **OBS** | Organizational Breakdown Structure - hierarchical company representation |
| **Blueprint** | Configuration template defining layout and modules for an object type |
| **Allocation** | Planned capacity committed to an investment |
| **Assignment** | Capacity assigned to specific tasks |
| **ETC** | Estimate to Complete - remaining effort or cost |
| **EAC** | Estimate at Completion - total projected effort or cost |
| **WBS** | Work Breakdown Structure - hierarchical task decomposition |
| **RAG** | Red/Amber/Green status indicator |
| **Portlet** | Reusable dashboard widget in Classic PPM |
| **Canvas** | Visual dashboard in Clarity Modern UX |
| **XOG** | XML Open Gateway - bulk data import/export tool |
| **GEL** | Generic Execution Language - scripting for processes |
| **NSQL** | Clarity query language for custom data providers |
| **Autoschedule** | Automatic task date calculation based on dependencies |
