## Tables in this subject area

### Tables documented in this vault
- [[BIZ_COM_PERIODS]] — This table stores the fiscal time periods set up by the user.
- [[CBK_ERRORS]] — This table stores the errors from a single chargeback run.
- [[CBK_ERROR_ARGUMENTS]] — This table stores the arguments of an error, such as the context required for the error message.
- [[CBK_GL_ACCOUNT]] — This table stores information on the GL chart of accounts.
- [[CBK_GL_ALLOCATION]] — This table stores information on the GL allocation combinations for the rules.
- [[CBK_GL_ALLOCATION_DETAIL]] — This table stores the time-varying distribution percentages to various GL accounts and departments.
- [[CBK_GL_TXNS]] — This table stores information on the GL transactions.
- [[CBK_GL_TXN_VALUES]] — This table stores the detail GL transaction values for the five supported currency types.
- [[CBK_INVOICE]] — This table stores the invoices generated by the system.
- [[CBK_SCALED_GL_TXNS]] — This table stores the GL transaction values scaled according to the Bill of IT (BOI) percentages.
- [[DEPARTMENTS]] — This table stores the department details for financial properties.
- [[ENTITY]] — The unique entity code.
- [[PAC_CHG_BATCH_HEADERS]] — This table stores batch posting information.
- [[PAC_FOS_RESOURCE_CLASS]] — This table stores resource class information.
- [[PAC_MNT_PROJECTS]] — This satellite project table is used for financial processing.
- [[PPA_BILLINGS]] — This table stores billing-related records.
- [[PPA_WIP]] — This table stores all primary transactions (Labor, Materials, Equipment, and Expense).
- [[PRCHARGECODE]] — This table stores data on charge codes.
- [[PRTYPECODE]] — This table stores the names of type codes.
- [[TRANSCLASS]] — This column refers to the code for the transaction class.
