# RQP_RELEASES

> [!info] Description
> This table stores the basic attributes of a release.

## Columns

| Column | Type | Null | Comment |
| --- | --- | --- | --- |
| `ID` | NUMERIC | NOT NULL | This column refers to the unique ID for the release generated by a sequence counter. |
| `NAME` | NVARCHAR | NOT NULL | This column refers to the name of the release. |
| `CODE` | NVARCHAR | NOT NULL | This column refers to the unique code identifying the release. |
| `TYPE` | NVARCHAR | NULL | This column refers to the release's type. |
| `HIGHLIGHTS` | NVARCHAR | NULL | This column refers to the description of the highlights of the relese. |
| `MANAGER_ID` | NUMERIC | NULL | This column refers to the ID of the manager of the release. |
| `STATUS` | NVARCHAR | NULL | This column refers to the status of the release. |
| `INVESTMENT_ID` | NUMERIC | NULL | This column refers to the ID of the investment to which the release belongs. |
| `ASSOC_PRJ_PRG_ID` | NUMERIC | NULL | This column refers to the ID of the project or program that is associated with the release. |
| `BGT_OP_COST` | NUMERIC | NULL | This column refers to the budgeted operating cost for the release. |
| `BGT_CAP_COST` | NUMERIC | NULL | This column refers to the budgeted capital cost of the release. |
| `BGT_EFFORT` | NUMERIC | NULL | This column refers to the amount of effort in hours or days budgeted for the release. |
| `BGT_SIZE` | NUMERIC | NULL |  |
| `START_DATE` | DATE | NULL | This column refers to the release's start date. |
| `END_DATE` | DATE | NULL | This column indicates the date of end of the release. |
| `MILESTONE_1` | DATE | NULL | This column refers to the date of a milestone in the release. |
| `MILESTONE_2` | DATE | NULL | This column refers to the date of a milestone in the release. |
| `MILESTONE_3` | DATE | NULL | This column refers to the date of a milestone in the release. |
| `MILESTONE_4` | DATE | NULL | This column refers to the date of a milestone in the release. |
| `MILESTONE_5` | DATE | NULL | This column refers to the date of a milestone in the release. |
| `ATTACHMENT` | NUMERIC | NULL | This column refers to the foreign key to release's attachments. |
| `CREATED_DATE` | DATE | NOT NULL | This column refers to the date and time this row was created. |
| `CREATED_BY` | NUMERIC | NOT NULL | This column refers to the resource ([[CMN_SEC_USERS]].ID) that created this row. |
| `LAST_UPDATED_DATE` | DATE | NOT NULL | The timestamp of when the record was last modified. |
| `LAST_UPDATED_BY` | NUMERIC | NOT NULL | This column indicates the ID of the resource who last modified this release. |
