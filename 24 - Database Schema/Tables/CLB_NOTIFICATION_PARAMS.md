# CLB_NOTIFICATION_PARAMS

> [!info] Description
> This table holds the parameters needed by the API to determine the referenced object and the flavor of notification.

## Columns

| Column | Type | Null | Comment |
| --- | --- | --- | --- |
| `ID` | NUMERIC | NOT NULL | The internal ID generated by a sequence. |
| `NOTIFICATION_ID` | NUMERIC | NOT NULL | The internal ID of the notification the parameter applies to. |
| `KEY_NAME` | NVARCHAR | NOT NULL | A key for the notification parameter for example NOTIFICATION_MESSAGE_KEY. |
| `VALUE` | NVARCHAR | NULL | A value for the notification parameter for example ID_TIMESHEET_RETURNED. |
