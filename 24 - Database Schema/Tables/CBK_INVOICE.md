# CBK_INVOICE

> [!info] Description
> This table stores the invoices generated by the system.

## Subject areas (ERDs)
- [[_MOC - Chargebacks ERD]]

## Columns

| Column | Type | Null | Comment |
| --- | --- | --- | --- |
| `ID` | NUMERIC | NOT NULL |  |
| `INVOICE_DATE` | DATE | NOT NULL | The date of the invoice. |
| `INVOICE_NUMBER` | NVARCHAR | NOT NULL | The unique system-generated invoice number. |
| `STATUS_CODE` | NUMERIC | NULL | Denotes the different statuses of the invoice. values are: - PRO-FORMA = Proforma - APPROVED = Approved - REJECTED = Rejected - RECALLED = Recalled - SUBMITTED = Submitted |
| `PERIOD_ID` | NUMERIC | NOT NULL | This column refers to [[BIZ_COM_PERIODS]].ID and indicates which fiscal time period the invoice is for. |
| `DEPARTMENT_ID` | NUMERIC | NOT NULL | This refers to [[DEPARTMENTS]].ID and indicates which department the invoice is for. |
| `CONSOLIDATED` | NUMERIC | NULL | This column indicates if the invoice consolidates the amounts from the sub-departments. |
| `IS_LOCKED` | NUMERIC | NULL | This column indicates if the invoice is locked. |
| `GENERATED_BY` | NUMERIC | NULL | The resource that generated this invoice. This refers to [[SRM_RESOURCES]].ID. |
| `CREATED_DATE` | DATE | NOT NULL | This column refers to the date and time this row was created. |
| `CREATED_BY` | NUMERIC | NOT NULL | This column refers to the resource ([[CMN_SEC_USERS]].ID) that created this row. |
| `LAST_UPDATED_DATE` | DATE | NOT NULL | The date and time this record was last updated. |
| `LAST_UPDATED_BY` | NUMERIC | NOT NULL | The resource that last updated this record. |
| `EXTERNAL_SOURCE_ID` | NUMERIC | NULL | If the invoices are imported from an external system, this field is populated with the ID of the external source. |
| `EXTERNAL_ID` | NVARCHAR | NULL | If the invoices are imported from an external system, this field is populated with the ID of the invoices from the external system. |
| `SCHED_JOB_ID` | NUMERIC | NULL | This refers to [[CMN_SCH_JOB_RUNS]].ID. The ID of the job that produced this invoice. |
