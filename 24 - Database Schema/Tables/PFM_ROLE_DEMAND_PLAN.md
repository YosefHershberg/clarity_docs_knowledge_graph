# PFM_ROLE_DEMAND_PLAN

> [!info] Description
> This table contains role demand changes in the context of a plan.

## Subject areas (ERDs)
- [[_MOC - Portfolios ERD]]

## Columns

| Column | Type | Null | Comment |
| --- | --- | --- | --- |
| `ID` | NUMERIC | NOT NULL | This column refers to the ID of the plan context |
| `PLAN_ID` | NUMERIC | NOT NULL | This column refers to the changes to the demand in the context of a plan |
| `DEMAND` | BINARY | NULL | The column contains changes to DEMAND in the context of a plan |
| `CREATED_DATE` | DATE | NOT NULL |  |
| `CREATED_BY` | NUMERIC | NOT NULL | This column contains the date the row was created |
| `LAST_UPDATED_DATE` | DATE | NOT NULL | This column contains the date that the row was last updated |
| `LAST_UPDATED_BY` | NUMERIC | NOT NULL | This column contains the user's name who last updated the row. |
