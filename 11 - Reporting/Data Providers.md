# Working with Data Providers

## Overview
Data Providers are the data sources that power reports and widgets in Clarity. They define what data is available for reporting.

## Key Concepts
- Data providers query Clarity data and make it available to reports
- Multiple providers can be used in a single report
- Providers support filtering and parameterization
- Shared data providers can be reused across reports

## Types
- Stock data providers (built-in)
- Custom data providers (created via NSQL queries)
