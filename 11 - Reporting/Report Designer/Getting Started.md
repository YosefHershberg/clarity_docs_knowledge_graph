# Working with Report Designer

## Overview

**Report Designer** is a visual tool for building custom reports and dashboards in Clarity. It is part of the Reporting workspace and supports iterative design, multi-tab layouts, and a variety of widget types for data visualization.

## Components

| Component | Description |
|-----------|-------------|
| **Canvas** | The report layout area where widgets are placed and arranged |
| **Widget Palette** | Library of available chart and data widgets |
| **Properties Panel** | Configure individual widget settings (Setup, Format, Style, Filter tabs) |
| **Filter Bar** | Define report-level, tab-level, and widget-level filters |

## Access Requirements

Users must have the following rights to access Report Designer:

- **Reporting - Navigate**
- **Report Designer - Navigate**
