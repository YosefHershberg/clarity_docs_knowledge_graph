# Working with Report Designer

## Overview
Report Designer is a visual tool for building custom reports and dashboards in Clarity.

## Components
- Canvas — The report layout area
- Widget Palette — Available chart and data widgets
- Properties Panel — Configure widget settings
- Filter Bar — Define report filters
