# Set Up a Cost and Rate Matrix

Configure the rate matrix that determines resource costs.

## Configuration
- Define rate rules by role, resource, location, department
- Set effective dates for rate changes
- Configure labor rates and non-labor rates
- Matrix lookup order determines which rate applies
