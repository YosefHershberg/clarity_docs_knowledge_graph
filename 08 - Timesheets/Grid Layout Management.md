# View and Manage Timesheets From Grid Layout

The Grid layout provides a comprehensive view of timesheet data across the organization.

## Features
- View all timesheets in a grid format
- Filter by resource, period, status
- Bulk approve or return timesheets
- Export timesheet data to CSV
