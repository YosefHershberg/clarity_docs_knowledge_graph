# Enter Hours and Submit a Timesheet

## Entering Time
1. Navigate to Timesheets from the main menu
2. Select the current timesheet period
3. Add tasks/investments to your timesheet
4. Enter hours per day for each task
5. Add notes if required

## Submitting
1. Review entries for accuracy
2. Click Submit
3. Status changes to Submitted
