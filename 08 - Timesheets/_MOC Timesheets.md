# Timesheets

Timesheets capture actual time spent by resources on investments.

## Notes in This Section
- [[Enter Hours and Submit]] — Enter time and submit timesheets (create methods, ETC, gauges, notes)
- [[Review and Approve]] — Manager review and approval workflow
- [[Grid Layout Management]] — View and manage timesheets from grid
- [[Frictionless Time]] — Unstaffed entry, Rally integration, auto-submit, auto-approve
- [[Advanced Timesheet Features]] — Enter Time for Others/Teams, attestation, charge codes, adjust posted, notifications, field security

## Timesheet Workflow
1. Enter Hours — Resources enter time against tasks/investments
2. Submit — Resource submits for approval
3. Review — Manager reviews submitted timesheets
4. Approve/Return — Manager approves or returns for correction
5. Post — Approved time feeds into financial actuals
