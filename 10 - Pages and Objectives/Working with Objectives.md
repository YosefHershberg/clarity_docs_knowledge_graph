# Working with Objectives

## Overview

Organizations can leverage the Objectives workspace to define objectives and key results (OKRs) that provide measurable outcomes to track progress. This helps with proactive decision-making and resource allocation.

## Key Concepts

### Objectives
- High-level strategic goals
- Aligned with organizational strategy
- Measurable through key results

### Key Results
- Specific, measurable outcomes
- Track progress toward objectives
- Time-bound targets

## Creating Objectives
1. Navigate to the Objectives workspace
2. Click + New Objective
3. Define the objective name, description, and owner
4. Add key results with target metrics
5. Link to investments and roadmap items

## Hierarchy
- Objectives can be organized hierarchically
- Parent objectives cascade to child objectives
- Key results roll up to parent objectives
