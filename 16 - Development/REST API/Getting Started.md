# Getting Started with Clarity APIs

## Overview
Clarity provides RESTful APIs for programmatic access to data and operations.

## Base URL
The API base URL follows the pattern: `https://<clarity-host>/ppm/rest/v1/`

## Key Concepts
- RESTful endpoints for CRUD operations
- JSON request/response format
- Token-based authentication
- Pagination support
- Field selection and filtering
