# Action Items (Mobile)

## Overview

Use Action Items in the Clarity Mobile app to review and act on approvals and workflow tasks.

## Prerequisites

You need Clarity Mobile App 3.2.3 and an appropriate Clarity server version.

- Clarity Mobile App 3.2.3 supports Clarity 16.0.3 and higher.
- Filtering Action Items by status requires Clarity 16.1.1 or higher.
- Action Item history fields (initiated process name, parent type) require Clarity 16.1.1 or higher.

> [!important]
> Starting in Mobile App 3.2.3, mobile-specific access rights are no longer used to show menu options. Users can see menu entries, but actions still depend on their Clarity access rights.

## Action Items Dashboard

The dashboard displays Action Items by received date and status.

1. Log in.
2. Tap `Menu`.
3. Tap `Action Items`.

### Dashboard sections

- Top tiles:
  1. Open: Action Items assigned to you with status Open.
  2. Pending: Action Items assigned to you with status Pending.
  3. Closed: Action Items assigned to you with status Closed.
- Middle tiles:
  1. Received this week: Open items received this week; percentage compares with last week.
  2. Received last week: Open items received last week; percentage compares with two weeks ago.
- Bottom carousel:
  - Up to 10 most recent open Action Items.
  - Swipe left or right to review cards.
  - Tap a card to open and act.

## View Action Item Cards

1. Log in.
2. Tap `Menu` > `Action Items`.
3. Tap any dashboard tile to open a pre-filtered list.

Each card shows:

- Subject
- Description
- Object and object name
- Status
- Priority
- Priority color

## Filter and Sort Action Items

You can:

- Search by Subject or Description from the search bar.
- Sort by date or alphabetically.
- Filter by one or more statuses.
- Filter by received range: last 7, 14, 30, 60 days, or anytime.
- Filter by one or more priorities.

1. Log in.
2. Tap `Menu` > `Action Items`.
3. Open any dashboard tile.
4. Tap the `Filter` icon.
5. Select options.
6. Tap `Back`.

> [!note]
> You can combine free-text search with additional filters. A filter counter indicates how many criteria are applied, and the page subtitle changes to `Filtered`.

## View Action Item History

History includes:

- Initiated date and time
- Initiating process or user
- Number of assignees
- Assignee details (name, avatar, email)
- Actions taken and action dates

1. Log in.
2. Tap `Menu` > `Action Items`.
3. Open an Action Item card.
4. Tap `View History`.

> [!note]
> Only five assignees are shown initially. Tap `See more` to view additional assignees.

## Refresh Action Item Pages

1. Log in.
2. Tap `Menu` > `Action Items`.
3. Pull down to refresh.

> [!note]
> Pull-down refresh works on both the dashboard and list pages.

## Act on Action Items

### Open status

1. Log in.
2. Tap `Menu` > `Action Items`.
3. Open an Action Item with status Open.
4. Select `Approve` or `Reject` and tap `Submit`.

A toast confirms the action, and the item moves to Pending or Closed.

### Pending status

1. Log in.
2. Tap `Menu` > `Action Items`.
3. Open an Action Item with status Pending.
4. Review your last action.
5. Select an action and tap `Submit`.

> [!note]
> Use `View History` to see who else acted on the item.

### Closed status

1. Log in.
2. Tap `Menu` > `Action Items`.
3. Open an Action Item with status Closed.

Closed items are view-only and show details plus your taken action.

> [!note]
> Use `View History` for full historical details.
