# Create Staffing Projections Using Scenarios

## Overview
Scenario Planning enables stakeholders to create multiple staffing projections for:
- Market demand changes
- Operational cost shifts
- Resource availability fluctuations

## Creating a Scenario
1. Navigate to the Staffing workspace
2. Click **Scenarios**
3. Create a new scenario
4. Adjust staffing levels
5. Compare scenarios side-by-side
