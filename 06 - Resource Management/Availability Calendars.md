# Manage Resource Availability Calendars

## Overview
Availability calendars determine a resource's working hours and days off.

## Configuration
- Set standard work hours per day
- Define weekends and holidays
- Create calendar exceptions for individual resources
- Assign calendars to resources

> [!note]
> Availability calendars affect autoschedule calculations and staffing metrics.
