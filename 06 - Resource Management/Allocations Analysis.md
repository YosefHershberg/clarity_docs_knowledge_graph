# Analyze Staffing by Allocations

Analyze staffing using allocation data to understand how resources are distributed across investments.

## Key Features
- View allocations by period (weeks, months, quarters)
- Compare allocation vs. availability
- Identify over-allocated resources
- Filter by OBS, skill, geography
- Create filter groups for complex filtering

## Partial Allocation
Partial allocation prevents overallocation by allowing percentage-based allocations within date ranges, enabling realistic multi-investment participation.
