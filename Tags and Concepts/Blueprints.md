# Blueprints

Blueprints are configuration templates that define the layout, modules, and behavior of each object type in Clarity.

## Key Capabilities
- Details Layout configuration
- Module enablement (Tasks, Staff, Financials, etc.)
- Business Rules for conditional logic
- Channels for routing users to different layouts
- Actions for process execution
