# Canvas

Canvas is a visual, widget-based dashboard tool used throughout Clarity for creating status reports and dashboards.

## Where Canvas is Used
- [[Status Reports|Investment Status Reports]] — Status report creation
- [[Canvas Dashboard|Investment Canvas]] — Investment dashboards
- [[_MOC Projects|Project Canvas]] — Project dashboards
- [[_MOC Roadmaps|Roadmap Canvas]] — Roadmap dashboards

## Widget Types
| Type | Description |
|------|-------------|
| **Status** | RAG indicators and status fields |
| **Text** | Rich text descriptions |
| **Data** | Auto-populated metrics from investment data |
| **Embedded** | Charts, reports, and external content |
