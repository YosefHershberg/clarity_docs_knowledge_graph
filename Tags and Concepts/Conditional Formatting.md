# Conditional Formatting

Conditional formatting in Clarity grids allows you to apply visual formatting (colors, icons) to cells based on data values.

## Use Cases
- Highlight overdue tasks in red
- Color-code investment health status
- Flag over-allocated resources
- Indicate budget variance thresholds

## Configuration
1. Open any [[Grids|grid]] view
2. Right-click a column header
3. Select Conditional Formatting
4. Define rules (conditions and formatting)
