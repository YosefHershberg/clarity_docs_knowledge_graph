# Getting Started with Projects

The **Projects** module in Clarity PPM is accessed from the main navigation menu. It lists all projects you have rights to view and provides multiple layouts for different work styles.

%%Source: p984-1001%%

---

## Accessing Projects

1. Click **Projects** in the left navigation rail.
2. The **Projects list** opens, showing all projects you have access to.
3. Use the **Search** bar at the top to filter by name.
4. Use [[Filters]] to narrow results by status, manager, OBS, dates, and custom fields.

> [!note] Access is controlled by your security rights and OBS membership. If a project does not appear, contact your administrator or see [[_MOC Administration]].

---

## The Projects List

The list view shows projects as rows with configurable columns. Key default columns include:

| Column | Description |
|--------|-------------|
| **Name** | Project name; click to open the project |
| **ID** | Unique project identifier |
| **Status** | Active, Inactive, or other configured statuses |
| **Manager** | The assigned project manager |
| **Start / Finish** | Planned project dates |
| **% Complete** | Rolled-up completion percentage |
| **Investment OBS** | Organizational unit the project belongs to |

### Sorting and Columns
- Click any **column header** to sort ascending/descending.
- Click the **column picker** (grid icon) to add or remove columns.
- Drag column headers to reorder.

> [!tip] Column configurations are saved per user. Each user can have their own column set.

## Project Layouts

Once inside a project, you can switch between layouts using the **layout switcher** at the top of the project detail view.

### Grid Layout
- Spreadsheet-style view of tasks, phases, and milestones.
- Add/edit tasks inline.
- See [[Tasks Phases Milestones]] for full details.

### Board Layout
- Kanban-style card view organized by status or phase.
- Drag cards between columns to change status.
- Configurable columns via board settings.

### Timeline Layout (Gantt)
- Horizontal bar chart showing task durations and dependencies.
- Zoom controls: days, weeks, months, quarters.
- Dependency lines drawn between linked tasks.

### Canvas Layout
- Fully customizable dashboard view.
- Add widgets: status fields, charts, team lists, custom text.
- See [[Canvas Dashboard]] for configuration details.

### Team Layout
- Focused view of project staff.
- Shows allocation, role, booking status, and dates.
- See [[Staffing]] for full details.

## Project Detail View

When you open a project, the **project header** always shows:

- Project name and ID
- Status indicator
- Manager name
- Key dates (start, finish)
- Quick-action icons (Edit, Follow, More options)

### Left Navigation (Project Modules)
Each project has a left-side navigation panel with modules:

| Module | Purpose |
|--------|---------|
| **Overview / Canvas** | Dashboard and summary |
| **Tasks** | Task management (grid/board/timeline) |
| **Team** | Staffing and resource allocation |
| **Financial Plans** | Cost, budget, and benefit plans |
| **Baselines** | Snapshot comparisons |
| **Risks** | Risk log |
| **Issues** | Issue tracker |
| **Changes** | Change request log |
| **Status Reports** | Progress reports |
| **Documents** | Attached files and links |
| **Checklists** | Quality checklists |
| **Audit** | Change history log |

> [!note] The modules visible depend on the **Blueprint** assigned to the project. Administrators configure blueprints in [[_MOC Administration]].

## Filtering and Searching Projects

### Quick Filter
- The **search bar** at the top of the projects list performs a name search.

### Advanced Filters
- Click **Filter** (funnel icon) to open the filter panel.
- Add conditions using AND/OR logic.
- Save filters as **Named Filters** for reuse.
- See [[Filters]] for full filter capabilities.

### My Projects vs. All Projects
- Toggle between **My Projects** (where you are the manager or team member) and **All Projects** using the view selector.

---

## Following a Project

Click the **Follow** (star/bell) icon on any project to receive notifications about changes. Followed projects appear in your **My Followed** smart list.
