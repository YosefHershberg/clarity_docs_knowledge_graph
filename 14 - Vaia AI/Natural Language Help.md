# Natural Language Help Assistant

## Natural Language Help Assistant

You can now use VaiaTM to ask various Clarity-related questions in natural language. VaiaTM is trained on the information from Clarity Technical Documentation and Support Knowledge base articles.
If your question does not include enough contextual information or is not relevant to Clarity, VaiaTM will display a message stating that it can only respond to questions associated with Clarity. Some key actions you can perform are:
- Ask VaiaTM questions in a natural language. For example: "What access rights do I need to delete a project in Clarity?"
- Mark a response as Useful or Not Useful using the Thumbs Up and Thumbs Down options.
- Provide additional feedback about a response by using the Feedback icon on the top-right corner of the response
dialog. Follow these steps:
1. Log into Clarity and click Administration, Feature Toggles.
2. Enable the Global Header Help Icon for the Information Center feature toggle.
3. Click the Question Mark icon next to the Notifications icon to open the Clarity Information Centre.
4. Click Assistant to launch VaiaTM.

In the top-left corner of the panel, click Home to access the Home panel. You can now access the complete documentation, videos, and training courses. NOTE Some key points to remember are: - Each question is considered as a separate thread.
VaiaTM does not preserve conversational context. - VaiaTM can be accessed using the new Clarity Information Center. The Information Center is turned off by default, but administrators need to toggle it on to make it available to users.
> [!warning]
> When you use VaiaTM , you are interacting with a generative AI service. AI-generated output may contain errors and unexpected results.
Do not include any personal data or confidential data in your prompts, and carefully review output before use.
