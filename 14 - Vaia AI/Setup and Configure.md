# Set up and Configure Vaia in Clarity

This note covers initial Vaia setup, configuration steps, and branding options.

## Setup Overview

1. **Activate the Vaia AI Service** — Enable the AI service in Clarity Administration.
2. **Configure the AI Service Provider** — Set up the integration with your chosen AI provider (e.g., OpenAI, Gemini).
3. **Validate the AI Model** — Confirm the configured AI model is working.
4. **Configure AI Agents** — Set up agents for Status Report Summarization, Roadmap Summarization, Data Generation/Revision/Review.
5. **Legacy Summaries** — Convert old Status Report and Roadmap summaries to Agents. Re-enable Roadmap summaries post-upgrade with a new report source.
6. **Monitor Consumption** — Track costs and usage via the **Queries** functionality in the Modern UX. Create queries to monitor token usage, service providers, and user activity.

---

## Brand Your AI

Use the **Branding** tab to customize how the AI capability is presented to users across Clarity.

- **Set the AI capability name:** Define how the assistant name appears across Clarity.
- **Upload a custom logo:** Use a dark-colored logo with a transparent background. Recommended format: PNG or GIF, 32px x 32px.

**Steps:**

1. Click **Administration** > **Vaia** > **Branding**.
2. Edit the **Name** field to your desired assistant name.
3. Click **Upload New Logo** and select your logo file.
4. Refresh the page to see the updated branding.

> [!TIP]
> If you delete the uploaded logo, the system reverts to the default AI logo automatically.
