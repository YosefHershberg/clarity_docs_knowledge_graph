# Manage Vouchers and Transactions

## Overview

Vouchers and transactions represent the actual financial activity recorded against investments. This workspace helps financial managers track, review, and reconcile financial data.

## Manage Unposted Transactions

Unposted transactions are financial entries awaiting approval or posting:
1. Navigate to the **Transactions** workspace
2. View unposted transactions in the grid
3. Review and validate transaction details
4. Post transactions to finalize them

## View Posted Transactions

Posted transactions are finalized financial records:
- Filter by investment, date range, transaction type
- View across multiple investments simultaneously
- Export to CSV for external analysis

## Transaction Types
- **Labor** — Time-based costs from timesheet entries
- **Non-Labor** — Expenses, materials, and other costs
- **Adjustments** — WIP adjustments and corrections
