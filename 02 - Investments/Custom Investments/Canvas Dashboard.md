# Configure Canvas Dashboard

## Overview

Configure the canvas to create a visual dashboard for the investment. Canvas provides drag-and-drop widget placement.

## Widget Types

| Type | Description |
|------|-------------|
| **Status** | RAG indicators and status fields |
| **Text** | Rich text blocks for descriptions |
| **Data** | Auto-populated metrics |
| **Embedded** | Charts, reports, and external content |

## Configuration

1. Open the investment
2. Navigate to the Canvas module
3. Click **Edit** to enter edit mode
4. Drag widgets from the palette onto the canvas
5. Configure each widget's data source and display
6. Click **Save** to publish

> [!note]
> Canvas layout is controlled by the [[Common Capabilities|Blueprint]] configuration. Administrators define available widget types.
