# Audit Module for Investments

## Overview

The Audit module tracks all changes made to an investment, providing a complete audit trail.

## What is Captured

- Field changes (old value → new value)
- Record creation and deletion
- User who made the change
- Date and time of change

## Using the Audit Grid

- Filter by Changed Date (default: last 10 days)
- Filter by attribute, user, or operation type
- Export to CSV for external analysis

> [!warning]
> The Audit module can contain large amounts of data. Always use date filters to improve performance.

## Configuration

Audit must be enabled in the [[Common Capabilities|Blueprint]] and configured in Classic PPM for specific attributes.
