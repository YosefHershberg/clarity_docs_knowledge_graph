# Investment Status Reports

## Overview

Status reports communicate investment progress to stakeholders.

## Working with Canvas

The Status Canvas provides a visual dashboard for creating status reports with configurable widgets:
- **Text Widgets** — Rich text sections for narratives
- **Status Widgets** — RAG indicators (Red/Amber/Green)
- **Data Widgets** — Auto-populated metrics from investment data
- **Embedded Widgets** — Charts and graphs

## Working with Reports Grid

The Reports Grid displays a list of all status reports for the investment:
- Filter by date range, status, author
- View report history
- Copy previous reports as templates

## Publishing

1. Create a status report (draft)
2. Fill in sections and widgets
3. Click **Publish** to make it available to stakeholders

> [!note]
> Published reports become read-only. Create a new report or copy an existing one for updates.
