# Hierarchy Module for Investments

## Overview

The Hierarchy module allows you to view and manage the hierarchical relationships of an investment within the broader organizational hierarchy.

## Key Features

- View parent and child investments
- Navigate the hierarchy tree
- Roll up financial and effort data
- View aggregated metrics across the hierarchy
