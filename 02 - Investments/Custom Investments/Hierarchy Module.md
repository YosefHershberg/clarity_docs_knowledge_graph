# Hierarchy Module for Investments

## Overview

The **Hierarchy** module allows you to view and manage the hierarchical relationships of an investment within the broader organizational hierarchy. It provides visibility into how investments are structured relative to parent and child objects across the portfolio.

## Key Features

- View **parent** and **child** investments in a tree structure
- Navigate up and down the hierarchy
- View **aggregated metrics** (financial totals, effort) rolled up from child to parent
- Analyze cross-investment performance from a single view

## Use Cases

- Verify that a custom investment is correctly positioned under its parent program or portfolio
- Review the rolled-up cost and effort of all child investments
- Navigate quickly to related investments within the hierarchy

> [!note]
> The Hierarchy module must be added to the investment's blueprint before it is visible. Contact your administrator to enable it.
