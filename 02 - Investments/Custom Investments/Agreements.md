# Agreements for Investments

## Overview

The **Agreements** module manages formal commitments between investments that are interdependent. Agreements allow stakeholders to review dependencies before roadmap planning activities or decisions that impact investment dates.

When an agreement is created between two investments, it appears in the **Agreements module** of both the provider and consumer investments.

> [!note]
> Administrators must manually add the **Agreements module** to each investment's blueprint. It is **not** automatically included in the default blueprint.

## Key Concepts

| Term | Description |
|------|-------------|
| **Provider** | The investment delivering the commitment or service |
| **Consumer** | The investment depending on the delivery |
| **Agreement Terms** | Defined deliverables, dates, and conditions of the commitment |

## Viewing Agreements

1. Open the investment
2. Navigate to the **Agreements** module
3. View all agreements where this investment is a provider or consumer
4. Click an **Agreement Name** to navigate to the full **Agreements workspace**

> [!note]
> You **cannot edit** agreements from within an investment's Agreements module. Navigate to the Agreements workspace to make changes.

## Creating an Agreement

Agreements are created and managed in the **[[Business Agreements]]** workspace:

1. Navigate to the **Agreements** workspace
2. Click **+ New Agreement**
3. Select the **provider** and **consumer** investments
4. Define the agreement terms, dates, and deliverables
5. Track agreement status through its lifecycle

## Access Rights

| Access Right | Description |
|-------------|-------------|
| Agreement - Create | Create new agreements |
| Agreement - Edit | Edit existing agreements |
| Agreement - Delete | Delete agreements |
| Agreement - Navigate | Access the Agreements workspace |
| Agreement - View | View agreement instances |
