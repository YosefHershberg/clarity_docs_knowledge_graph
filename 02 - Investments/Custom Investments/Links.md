# Investment Links

## Overview

The **Links** module connects an investment to related investments, external resources, or documents, providing quick navigation between related content.

## Link Types

| Type | Description |
|------|-------------|
| **Investment Links** | Link to other investments within Clarity |
| **URL Links** | Link to external websites or online resources |
| **Document Links** | Link to uploaded documents stored in Clarity |

## Creating a Link

1. Open the investment and navigate to the **Links** module
2. Click **+ Add Link**
3. Select the **link type** from the dropdown
4. Select or enter the **target** (investment, URL, or document)
5. Add an optional **description** for context
6. Click **Save**

## Managing Links

- Edit or delete links by selecting a record in the Links grid and using the right-click menu or **Actions** menu
- Links are visible to all users with view access to the investment
