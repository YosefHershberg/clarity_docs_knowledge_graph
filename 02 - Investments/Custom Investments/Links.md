# Investment Links

## Overview

Links connect an investment to related investments, external resources, or documents.

## Types

- **Investment Links** — Link to other investments in Clarity
- **URL Links** — Link to external websites or resources
- **Document Links** — Link to uploaded documents

## Creating a Link

1. Open the investment's **Links** module
2. Click **+ Add Link**
3. Select link type and target
4. Add a description
