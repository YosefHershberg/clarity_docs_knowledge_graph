# Investment Conversations

## Overview

The **Conversations** module enables team communication within the context of an investment, keeping all discussions tied to the relevant work item.

## Features

- Post messages visible to all investment team members
- Reply to existing conversation threads
- **@mention** specific team members to notify them
- Attach files and links to messages
- View the full conversation history

## Access Rights for Conversation Actions

| Action | Access Right Required |
|--------|-----------------------|
| View a message | View access to the investment |
| Reply to a message | View access to the investment |
| Attach files | View access to the investment |
| Navigate to the investment | \<Investment\> - Navigate |
