# Staff a Custom Investment

## Overview

Staffing an investment means assigning resources (people) or roles (placeholders) to the investment. The Staff module provides tools for managing team composition, allocation, and booking types.

## Adding Resources

1. Open the investment and navigate to the **Staff** module
2. Click **+ Add** to add a resource or role
3. Search for the resource by name, role, or skill
4. Set the **allocation** (percentage or hours)
5. Set the **booking type**: Hard (committed) or Soft (tentative)

## Booking Types

| Type | Description |
|------|-------------|
| **Hard** | Committed allocation — resource is confirmed |
| **Soft** | Tentative allocation — resource is planned but not confirmed |
| **Mixed** | Both hard and soft bookings on the same investment |

## Allocation Values

Allocation values define how much of a resource's capacity is committed:
- View allocations by **FTE**, **Hours**, or **Percentage**
- Configure time-scaled allocation values across periods
- Use the allocation curve to distribute effort over time

## Replace and Delete Staff

### Replace a Staff Member
1. Select the staff member to replace
2. Click **Actions** > **Replace**
3. Search for and select the replacement resource
4. Choose whether to transfer allocations and assignments

### Delete a Staff Member
1. Select the staff member
2. Click **Delete**
3. Confirm removal

> [!warning]
> Deleting a staff member removes their allocations and may affect task assignments.

## Access Rights

| Right | Description |
|-------|-------------|
| Investment - Edit Staff | Add/edit staff on the investment |
| Investment - View Staff | View staff members |
| Resource - Soft Book | Create soft bookings |
| Resource - Hard Book | Create hard bookings |
