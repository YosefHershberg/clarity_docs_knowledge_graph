# Investment Phases, Milestones, Tasks, and To-Do Items

## Task Types

| Type | Description | Icon |
|------|-------------|------|
| **Phase** | Container for tasks; rolls up dates and effort | Folder |
| **Milestone** | Zero-duration checkpoint; marks a key date | Diamond |
| **Task** | Work item with start/finish, effort, and assignments | Rectangle |
| **To-Do** | Simple personal checklist item | Checkbox |

## Managing Tasks in Grid Layout

- Add tasks using **+** in the toolbar
- **Indent/Outdent** to create phase hierarchy
- Inline edit: name, dates, status, % complete, ETC
- **Drag and drop** to reorder tasks within a phase
- Configure columns to show relevant task attributes

## Managing Tasks in Board Layout

- Tasks display as cards in columns (grouped by status or custom field)
- **Drag cards** between columns to update status
- Click a card to open task details
- Configure card fields and column groupings

## Managing Tasks in Timeline Layout

- Visual Gantt-style view of task schedule
- **Drag task bars** to adjust dates
- Create **dependencies** by dragging between task bars
- View the **critical path** to identify schedule-driving tasks
- Use **Autoschedule** to automatically calculate dates based on dependencies

> [!tip]
> See [[Autoschedule]] for details on automatic schedule calculation.

## Standard Features Across Layouts

- **% Complete** — Track task progress
- **ETC (Estimate to Complete)** — Remaining effort estimate
- **Assignments** — Assign resources/roles to tasks
- **Dependencies** — Finish-to-Start, Start-to-Start, etc.
- **Constraints** — Must Start On, Must Finish On, etc.
