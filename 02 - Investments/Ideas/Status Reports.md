# Manage Status Reports for Ideas

## Working with Idea Canvas
- Configure visual status dashboards with widgets
- Text, status, data, and embedded widgets available
- Use RAG indicators for quick status communication

## Working with Reports Grid
- View all status reports in a grid format
- Filter by date, status, author
- Copy previous reports as templates

## Publishing Status Reports
1. Create a draft report
2. Fill in sections and update widgets
3. Click **Publish** to distribute to stakeholders
