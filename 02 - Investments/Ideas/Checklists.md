# Creating and Managing Idea Checklists

## Overview

Checklists for ideas work the same as for other investments — see [[Checklists|Custom Investment Checklists]] for detailed procedures including how to create checklists, track progress, and compare checklists against To-Dos and Risks.

## Use Cases for Idea Checklists

Ideas have specific checklist use cases that align with the idea evaluation lifecycle:

| Use Case | Description |
|----------|-------------|
| **Innovation Review Criteria** | Define evaluation criteria for new ideas |
| **Feasibility Assessment Steps** | Track feasibility study tasks and completion |
| **Approval Gate Requirements** | List items required before advancing to the next gate |
| **Due Diligence Items** | Steps to complete before converting the idea to a project |

## Tracking Progress

- Completion percentage auto-calculates as items are checked off
- Filter checklist items by: **All**, **Completed**, **Pending**
- Checklists are visible to all team members with access to the idea
