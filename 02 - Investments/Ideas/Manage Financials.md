# Manage Idea Financials

## Overview

The Financials module for Ideas provides the same financial planning capabilities as other investment types, enabling organizations to evaluate the financial viability of ideas before committing to full projects.

## Plan Types

| Plan | Purpose |
|------|---------|
| **Cost Plan** | Estimate the cost of developing the idea |
| **Budget Plan** | Approved spending for idea development |
| **Benefit Plan** | Expected returns if the idea becomes a project |
| **Forecast** | Updated cost projection |

## Working with Cost Plans
1. Navigate to the idea's **Financials** module
2. Select **Cost Plans** tab
3. Click **+ New** to create a plan
4. Add line items by resource, role, or expense type
5. Enter planned amounts per period

## Working with Budget Plans
- Promote a cost plan to become the approved budget
- Track actual spending against budget

## Working with Benefit Plans
- Model expected returns or revenue
- Compare benefits against costs for ROI analysis
