# Manage Risks, Issues, and Changes for Ideas

Risk, issue, and change management for ideas follows the same patterns as [[Risks Issues Changes|Custom Investment RIC]].

## Key Differences for Ideas
- Risks identified during idea development help evaluate feasibility
- Issues may influence the approve/reject decision
- Change requests can modify the idea scope before conversion
