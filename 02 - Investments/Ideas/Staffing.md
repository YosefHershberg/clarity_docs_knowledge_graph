# Staff an Idea

Staffing ideas works the same way as staffing other investments. See [[Staffing|Custom Investment Staffing]] for detailed procedures.

## Key Points
- Add named resources or placeholder roles
- Set allocation percentages and booking types (Hard/Soft)
- Replace or delete staff members as needed
- Access rights control who can modify staffing
