# Getting Started with Ideas

## Overview

The Ideas workspace serves as a centralized hub for capturing, developing, and approving new business initiatives before they evolve into full-scale projects or investments. It helps organizations systematically evaluate their innovation pipeline.

## Prerequisites

| Access Right | Description |
|-------------|-------------|
| Idea Management - Navigate | Access the Ideas workspace |
| Ideas - Create | Create new ideas |
| Resource - View Ideas - All | View all idea resources |
| Resource - Approve Ideas - All | Approve ideas |

## Creating Ideas

### From Scratch
1. Navigate to the **Ideas** workspace
2. Click **+** in the Grid layout
3. Fill in required fields
4. Click **Save**

### From Template
1. Click **+ New from Template**
2. Select an idea template
3. Templates ensure consistency across the pipeline

### CSV Import (Beta)
- Import multiple ideas from a CSV file
- Download the template, populate data, upload

### Copy an Idea
- Open the source idea > **Actions** > **Copy**
- Select elements to carry over

## Converting Ideas to Projects

One of the key workflows is converting approved ideas into projects:
1. Open the approved idea
2. Click **Actions** > **Convert to Project** (or Custom Investment)
3. Select a project template to use
4. Map idea fields to project fields
5. The idea's data transfers to the new project

> [!tip]
> Use templates during conversion to ensure the new project inherits the correct blueprint configuration.

## Blueprint Configuration

The Standard Idea blueprint allows administrators to configure layouts:
- Add relevant sections and fields to the Details layout
- Enable modules: Tasks, Staff, Financials, Risks, Issues, Changes
- Support business rules and channels

## Navigating Layouts

Ideas support the same layouts as other investments:
- **[[Grids|Grid]]** — Detailed spreadsheet-like view
- **[[Board Layout|Board]]** — Kanban-style drag-and-drop
- **Timeline** — Chronological view
- **Tiles** — Visual card overview
