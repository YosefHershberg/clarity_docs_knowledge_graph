# Configure Canvas to Manage the Idea Dashboard

Configure a visual canvas dashboard for ideas using drag-and-drop widgets. See [[Canvas Dashboard|Custom Investment Canvas Dashboard]] for detailed configuration steps.

## Available Widget Types
- Status widgets (RAG indicators)
- Text widgets (rich text descriptions)
- Data widgets (auto-populated metrics)
- Embedded widgets (charts, external content)
