# Configure Canvas to Manage the Idea Dashboard

## Overview

The Canvas module provides a visual, drag-and-drop dashboard for ideas. It works the same way as the Canvas module for other investment types — see [[Canvas Dashboard|Custom Investment Canvas Dashboard]] for the full configuration procedure.

## Available Widget Types

| Widget | Description |
|--------|-------------|
| **Status Widgets** | RAG (Red/Amber/Green) status indicators |
| **Text Widgets** | Rich text descriptions and narratives |
| **Data Widgets** | Auto-populated metrics from idea data |
| **Embedded Widgets** | Charts and external content |
| **Number Tile** | Displays a specific count or metric |
| **Progress Ring** | Ring chart showing progress toward a target |
| **Pie Chart** | Grouped pie chart by a selected attribute |
| **Bar Chart** | Bar graph (horizontal or vertical) |
| **Table Widgets** | Data grids for Tasks, RICs, Staff, Financials, Status |

## Prerequisites

- You have **View** access to the idea
- The idea is associated with a blueprint that includes the **Canvas** module
