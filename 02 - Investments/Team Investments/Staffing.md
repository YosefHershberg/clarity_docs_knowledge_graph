# Staff a Team Investment

## Overview

Team staffing follows the same patterns as other investment types — see [[Staffing|Custom Investment Staffing]] for the full procedure including adding resources, setting allocation percentages, booking types, replacing, and deleting staff.

## Team-Specific Considerations

Teams differ from projects in that staffing is typically **long-term and persistent** rather than project-bound:

| Consideration | Description |
|---------------|-------------|
| **Persistent allocation** | Resources may be **fully allocated (100%)** to a single team long-term |
| **Mixed bookings** | Supports partial team membership via mixed Hard/Soft bookings |
| **Capacity tracking** | Allocation values track team capacity over rolling time periods |
| **Stable composition** | Team membership changes less frequently than project staffing |

## Booking Types

| Type | Use Case |
|------|----------|
| **Hard** | Core team members with full committed allocation |
| **Soft** | Temporarily supporting the team or being evaluated for permanent assignment |

## Access Rights

| Right | Description |
|-------|-------------|
| Investment - Edit Staff | Add or edit staff on the team investment |
| Investment - View Staff | View team members |
| Resource - Soft Book | Create soft bookings |
| Resource - Hard Book | Create hard bookings |
