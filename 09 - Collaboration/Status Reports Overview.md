# Manage Status Reports

Status reports communicate investment progress to stakeholders.

## Report Components
- RAG Status (Red/Amber/Green indicators)
- Narrative Sections
- Key Metrics (auto-populated)
- Canvas Widgets (visual dashboards)

## Workflow
1. Create a draft report
2. Edit sections and widgets
3. Publish to stakeholders (becomes read-only)
4. Copy published report as template for next period
