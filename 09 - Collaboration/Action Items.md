# Action Items in My Workspace

Action items are tasks or activities requiring your attention.

## Viewing Action Items
1. Navigate to My Workspace then Action Items
2. View pending items in a grid
3. Filter by type, priority, or due date

## Types
- Process approvals
- Timesheet reviews
- Resource requests
- Document reviews
